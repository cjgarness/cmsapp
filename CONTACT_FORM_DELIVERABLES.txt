QUICK START (5 STEPS)
═══════════════════════════════════════════════════════════════════════════════

1. Run migrations:
   ./migrate.sh dev migrate

2. Initialize configuration:
   docker-compose -f docker-compose.dev.yml exec web python manage.py init_contact_config

3. Configure email backend (see CONTACT_FORM_CONFIG.md):
   Set EMAIL_BACKEND, EMAIL_HOST, etc. in settings.py or .env

4. Update admin settings:
   Visit http://localhost:8000/admin/contact/contactconfiguration/

5. Test the form:
   Navigate to http://localhost:8000/contact/
